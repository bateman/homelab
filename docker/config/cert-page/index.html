Homelab CA Certificate

Install the Root CA to trust all *.home.local services

Apple Devices Recommended

iPhone, iPad, Mac — installs the CA certificate with a guided flow.

Install Profile

Other Platforms

Windows, Linux, Android, Firefox — download the CA certificate file.

Download CA Certificate
iPhone / iPad
  1. Tap Install Profile above (use Safari)
  2. A prompt appears: Profile Downloaded — tap Close
  3. Open Settings → General → Profiles
  4. Tap the downloaded profile → Install → enter passcode
  5. Go to Settings → General → About → Certificate Trust Settings
  6. Enable Full Trust for Homelab Root CA
macOS
  1. Tap Install Profile above, or download ca.crt
  2. If using the profile: follow the prompts in System Settings
  3. If using ca.crt: double-click the file → Keychain Access opens
  4. Select System keychain → Add
  5. Find Homelab Root CA in the list → double-click
  6. Expand Trust → set Always Trust
  7. Or via terminal: 
    sudo security add-trusted-cert -d -r trustRoot \
    -k /Library/Keychains/System.keychain ca.crt
Windows
  1. Download ca.crt above
  2. Double-click the file → Install Certificate
  3. Select Local Machine → Next
  4. Place all certificates in the following store → Browse
  5. Select Trusted Root Certification Authorities
  6. Finish → restart browser
Linux (Chrome / Chromium)
  1. Download ca.crt above
  2. Import into NSS database: 
    certutil -d sql:$HOME/.pki/nssdb -A \
    -t "C,," -n "Homelab Root CA" -i ca.crt
  3. Restart browser
Firefox (all systems)
  1. Download ca.crt above
  2. Firefox: Settings → Privacy & Security → Certificates → View Certificates
  3. Authorities tab → Import
  4. Select ca.crt → check Trust this CA to identify websites
Android
  1. Download ca.crt above
  2. Settings → Security → Encryption & credentials
  3. Install a certificate → CA certificate
  4. Select the downloaded file
One-time setup. Once the CA is trusted, all *.home.local services (Sonarr, Radarr, etc.) will show a valid certificate — no more browser warnings. The server certificate can be rotated without repeating this process.
